# 9-Edge Screening — 朋友使用指南

房主會俾你 **一條 Streamlit Cloud link**（例如 `https://xxx.streamlit.app`）。  
用 Chrome / Edge 開就得，**唔使** 安裝 TradingView，**唔使** 連房主部機。

---

## 快速步驟

1. 開房主俾嘅 **Cloud link**
2. 主畫面輸入 **股票代號**（例如 `WOLF`、`TSM`、`NVDA`）
3. 撳 **「🔍 分析股票」** — 等約 10–30 秒
4. 主畫面睇完整 9-edge 報告（Grade、Setup、價格情景等）

**唔使上傳 CSV。**

---

## 睇 Batch 選股結果

Sidebar **報告庫** → 揀 `SCREENER_..._summary` → **載入**  
→ 睇 A/B 級、潛力榜、完整排名

或主畫面撳 **「載入最新 Screener 摘要」**。

---

## 進階（一般唔使）

Sidebar **進階：上傳 CSV** — 只有當你想用 TradingView 匯出嘅精準 CSV 先需要。

---

## 常見問題

**Q：同房主本機 TV 分析有咩分別？**  
A：Cloud 用 **yfinance** 拉數，方便快速查任何美股；房主本機可用 TradingView 拉更精準數據。

**Q：分析完 report 刷新後冇咗？**  
A：Cloud 係暫存；請 copy 或 download 內容。
